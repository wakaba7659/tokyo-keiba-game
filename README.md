# 東京競馬場 — TOKYO RACECOURSE GAME 🏇

ブラウザで遊べる本格競馬券ゲーム。東京競馬場を3Dで再現しています。

**▶ 遊ぶ:** https://wakaba7659.github.io/tokyo-keiba-game/

## 特徴

- **東京競馬場を実データで再現** — 左回り・芝1周2083.1m・直線525.9m・高低差（だんだら坂含む）を3Dモデル化
- **7券種 × 4方式** — 単勝・複勝・ワイド・馬連・馬単・三連複・三連単／通常・ボックス・ながし・フォーメーション
- **マークシート式の購入UI** — 実際のJRAマークシート風
- **本馬場入場・返し馬** — 各馬の走りっぷりと気配を中継カメラで確認してから購入
- **レースエンジン** — 能力・調子・脚質（逃げ/先行/差し/追込）・芝ダート適性・坂でのスタミナ消耗を反映。芝は差しが決まりやすく、ダートは前が止まりにくい
- **実況カメラ** — 俯瞰中継 → 直線ズーム → ゴール正面（着順が分かる真横アングル）→ 勝ち馬追走。僅差はスローモーション
- **12レース制の開催** — 毎日の11Rは実在GI（フェブラリーS、日本ダービー、ジャパンカップなど7レースを巡回）。GI専用ファンファーレ付き
- **ターフビジョン・ウイナーズサークル** — レース後の雰囲気まで再現

## 遊び方

1. 所持金を設定して入場（100円単位）
2. 出走表・パドック・本馬場入場で馬の調子をチェック
3. マークシートで馬券を購入（1点あたり100円から）
4. 「発走！」でレース観戦 — 的中すれば払戻！

## 技術

HTML/CSS/JavaScript（`index.html`）。3D描画は [Three.js](https://threejs.org/)（CDN）。
オフライン環境では2D簡易表示に自動フォールバックします。
騎手名リストは `jockeys.js` に外出し（編集はこのファイルのみでOK）。

`_test_logic.js` はレースエンジン・オッズ計算の検証ハーネスです（`node _test_logic.js`）。
